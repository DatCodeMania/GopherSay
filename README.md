# GopherSay

A fun CLI tool that I wrote in my journey of learning Go.

![Example](example.png)

## Usage

### Linux
```
git clone https://github.com/DatCodeMania/GopherSay.git
cd GopherSay
go build -o GopherSay .
./GopherSay "Message"
```

### Windows
```
git clone https://github.com/DatCodeMania/GopherSay.git
cd GopherSay
go build -o GopherSay.exe .
./GopherSay "Message"
```
